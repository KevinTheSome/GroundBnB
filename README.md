# GroundBnB<br/>
GroundBnB is the next AirBnB using Laravel and React<br/>

## Run it for doing:<br/>
Backend:<br/>

```
composer i
```

```
php artisan migrate:fresh
```

```
php artisan storage:link
```
```
php artisan serve
```
```
php artisan storage:link
```
Frontend:<br/>
```
npm i
```
```
npm run dev
```
